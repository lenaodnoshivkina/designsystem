О дизайн системе

Цели и задачи

#1

Цели

Систематизировать дизайн для творческого диджитал-объединения «CURSA» с помощью создание взаимосвязанных базовых элементов (дизайн-токены и библиотеки компонентов) и прийти к единому полю взаимосвязанных продуктов, таких как страница объединения, страницы-анонсов мероприятий и обучающих материалы. То есть создать удобную экосистему продуктов, которая будет понятна разработчикам и пользователям.

#2

Задачи

  • Определить философию компании;

  • Определить элементы фирменного стиля;

  • Создать наборы готовых компонентов и описать их;

  • Создать гайдлайны по использованию фирменного стиля;

  • Сформировать документацию по техническим особенностям реализации компонентов.
